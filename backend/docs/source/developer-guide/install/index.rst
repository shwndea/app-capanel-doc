Install
================================================================

Prerequisites
-----------------------------
- Python 3.10+ required. Python 3.14 recommended.
- uv or pip, and a working virtual environment tool. Only uv is covered in this documentation.
- PostgreSQL
- Node.js
- Docker (optional)

Choose your platform to install the required dependencies.
