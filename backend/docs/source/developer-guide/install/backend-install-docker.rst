Docker
################################################################
The frontend and backend can be run together with Docker.

Windows and Mac
************************************
Install Docker on Windows and Mac with `Docker Desktop <https://docs.docker.com/desktop/>`_.

Linux
************************************
Install Docker on Linux with `Docker Engine <https://docs.docker.com/engine/install//>`_.

Next step
=====================================================

.. code-block:: bash

   docker compose up -d

See `Run the application <https://opensacorg.github.io/app-capanel-doc/developer-guide/run-application>`_.
