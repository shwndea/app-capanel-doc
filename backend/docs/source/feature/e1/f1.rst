Overview
--------

The Performance Cards feature is built using six core UI components that work together to display California Dashboard accountability data in an accessible, user-friendly format.

Components
----------

This feature uses the following components:

1. **StatusLevelIndicator**: Color-coded performance level (Blue, Green, Yellow, Orange, Red)
2. **ChangeLevelIndicator**: Year-over-year performance change indicator
3. **MetricCard**: Container component that holds all metric information
4. **StudentGroupBreakdown**: Disaggregated data table by student demographics
5. **TrendChart**: Multi-year performance visualization
6. **DataSuppressionNotice**: Privacy protection message when data cannot be displayed
